⚖️ Hukum Dasar (1)
Hukum Newton II
ΣF = m × a = m × y''
"Jumlah gaya = massa × percepatan"
Hukum Hooke (Pegas)
F_pegas = -k × y
"Gaya pegas sebanding dengan simpangan tetapi berlawanan arah"

Keterangan: m (kg), y'' (m/s²), k (N/m), y (m)

🔌 Hukum Dasar (2)
Hukum Kirchhoff (Tegangan)
ΣV = 0   atau   RI + L·dI/dt + (1/C)∫I dt = E(t)

Keterangan komponen:

  • RI = tegangan pada resistor
  • L dI/dt = tegangan pada induktor
  • (1/C)∫I dt = Q/C = tegangan pada kapasitor
  • E(t) = sumber tegangan
I = dQ/dt   (Kuat arus = laju perubahan muatan)
🌀 Osilasi Bebas Tak Teredam

Sistem: Massa + Pegas (tanpa redaman)

  • Gaya pegas: -ky
  • Tidak ada gaya redaman
  • Tidak ada gaya luar
Karakteristik:
  • Energi mekanik kekal
  • Osilasi terus menerus
  • Amplitudo tetap
📝 Penurunan PD Tak Teredam
Langkah 1: ΣF = my'' → -ky = my''
Langkah 2: my'' + ky = 0
Langkah 3: y'' + (k/m)y = 0
Misalkan ω₀² = k/m, maka:
y'' + ω₀² y = 0
📐 Solusi Umum Tak Teredam
Persamaan karakteristik: λ² + ω₀² = 0
Akar: λ = ± i ω₀
Solusi umum:
y(t) = A cos(ω₀ t) + B sin(ω₀ t)
Bentuk alternatif:
y(t) = R cos(ω₀ t - φ)
dengan R = √(A²+B²) = Amplitudo
φ = arctan(B/A) = Sudut fase
📌 Contoh Soal 1: Tak Teredam

Soal: Bola besi berat W = 70 N meregangkan pegas 5 cm. Ditarik ke bawah 10 cm lalu dilepas (v₀=0). Tentukan y(t) dan frekuensi.

Penyelesaian:
k = W/Δs = 70/0,05 = 1400 N/m
m = W/g = 70/9,8 = 7,14 kg
ω₀ = √(k/m) = √196 = 14 rad/s
y(t) = A cos 14t + B sin 14t
y(0)=0,1 → A=0,1
y'(0)=0 → B=0
Hasil: y(t) = 0,1 cos 14t
f = ω₀/(2π) = 14/(2π) ≈ 2,23 Hz
🌊 Osilasi Bebas Teredam

Massa + Pegas + Peredam (c = konstanta redaman)

  • Gaya pegas: F₁ = -ky
  • Gaya redaman: F₂ = -c y'
Hukum Newton II:
-ky - c y' = m y''
m y'' + c y' + k y = 0
📊 Tiga Kasus Redaman
KasusSyaratSolusi
Overdampedc² > 4mky = C₁e^{λ₁t} + C₂e^{λ₂t}
Critically Dampedc² = 4mky = (C₁ + C₂t)e^{λt}
Underdampedc² < 4mky = e^{at}(C₁ cos bt + C₂ sin bt)
a = -c/(2m)    b = √(4mk-c²)/(2m)
📌 Contoh Soal 2: Teredam Kritis

Soal: Bola besi sama (W=70N, k=1400 N/m, m=7,14 kg) dengan redaman c=200. y(0)=0,1 , y'(0)=0. Tentukan y(t).

PD: y'' + 28y' + 196y = 0
λ² + 28λ + 196 = 0
λ₁,₂ = -14 (akar kembar)
Solusi umum: y(t) = (C₁ + C₂t)e^{-14t}
y(0)=0,1 → C₁ = 0,1
y'(0)=0 → C₂ = 1,4
Hasil: y(t) = (0,1 + 1,4t)e^{-14t}
🔋 Rangkaian RLC Seri
  • Resistor (R): E_R = RI
  • Induktor (L): E_L = L dI/dt
  • Kapasitor (C): E_C = (1/C)Q
Hukum Kirchhoff:
L·dI/dt + RI + (1/C)∫I dt = E(t)
Turunkan terhadap t:
L I'' + R I' + (1/C) I = E'(t)
📌 Contoh Soal 3: RLC

Soal: R=100Ω, L=0,1H, C=1mF, tanpa sumber. I(0)=3mA, I'(0)=0. Tentukan I(t).

PD: I'' + (R/L)I' + (1/(LC))I = 0
I'' + 1000I' + 10000I = 0
λ² + 1000λ + 10000 = 0
λ₁ ≈ -10,1 , λ₂ ≈ -989,9
I(t) = C₁e^{-10t} + C₂e^{-990t}
I(0)=0,003 , I'(0)=0
C₁ ≈ 0,00303 , C₂ ≈ -3,09×10^{-5}
Hasil: I(t) = 0,00303 e^{-10t} - 3,09×10^{-5} e^{-990t}

Tak teredamc=0R=0A cos ωt + B sin ωt
Teredam kurangc²<4mkR²<4L/Ce^{at}(C₁ cos ω_d t + C₂ sin ω_d t)
Teredam kritisc²=4mkR²=4L/C(C₁+C₂t)e^{at}
Teredam lebihc²>4mkR²>4L/CC₁e^{λ₁t}+C₂e^{λ₂t}

📝 Langkah Umum Penyelesaian
  1. Identifikasi sistem (pegas/redaman/rangkaian)
  2. Tulis hukum fisika yang relevan
  3. Bentuk persamaan diferensial orde 2 homogen
  4. Tentukan koefisien (k, m, c, R, L, C)
  5. Tulis persamaan karakteristik
  6. Cari akar-akar λ
  7. Tentukan kasus (tak teredam/teredam kurang/kritis/lebih)
  8. Tulis solusi umum sesuai kasus
  9. Terapkan syarat awal
  10. Interpretasi hasil
